## 🎮 Как работает

| Сайт | До | После |
|------|----|-------|
| **YouTube** | 🅾️ Логотип | `14<span style="color:#FF0000">:45</span>` |
| **VK.com** | ➖ Ничего | `🕐 14:23:45` в левом нижнем углу |

**✨ Особенности:**
- 🔴 Красные секунды на YouTube
- ⚡ Обновление каждую секунду
- 📱 Работает с защитой YouTube 2026
- 🔄 Автообновление скрипта
