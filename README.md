# Часы для ВК и YouTube
Скрипт для Tempermonkey, добавляющий часы в левый нижний угол ВК, а также заменяющий на часы логотип YouTube
[![Greenkeeper badge](https://img.shields.io/badge/Install-Tampermonkey-brightgreen)](https://raw.githubusercontent.com/ViktorKoval5734/Clock_for_VK_and_YouTube/main/Часы-в-VK-и-YouTube.user.js)

## 🚀 Установка
1. Установите [Tampermonkey](https://www.tampermonkey.net/)
2. Нажмите кнопку **Install** выше
3. Готово! 🔴
